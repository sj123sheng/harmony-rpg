# Harmony RPG MVP QA 清单

## 设备

- HarmonyOS 手机或模拟器。
- 横屏方向。

## 移动

- 玩家可以通过左侧摇杆连续移动。
- 玩家会根据移动方向面向左、右、上或下。
- 松开摇杆后玩家停止移动。

## 碰撞

- 玩家不能穿过左侧竹林边界。
- 玩家不能穿过顶部竹林边界。
- 玩家不能进入茶铺建筑。
- 玩家不能穿过古井。
- 玩家不能穿过灵灯底座。
- 玩家不能穿过底部围栏离开场景。

## 交互

- 靠近守灯人时出现交互按钮。
- 靠近茶铺老板时出现交互按钮。
- 靠近阿梨时出现交互按钮。
- 靠近井边灵石时出现交互按钮。
- 靠近村口灵灯时出现交互按钮。
- 多个对象都在附近时，选择距离最近的对象。

## 任务

- 守灯人会开启任务。
- 任务开启后，茶铺老板会给出古井线索。
- 获得茶铺线索后，井边灵石会推进任务。
- 找到灵石后，村口灵灯会完成任务。
- 任务完成后，灵灯显示为点亮状态。

## UI

- 摇杆固定在左下角。
- 交互按钮固定在右下角。
- 对话框不会遮挡摇杆或交互按钮。
- 目标手机上，对话文本能完整显示在对话框内。

## 第二阶段视觉升级

- 地图能清楚区分草地、土路、竹林、茶铺、古井和底部围栏。
- 玩家和 3 个 NPC 都具备身体、头部、阴影和基础朝向提示。
- 靠近 NPC、井边灵石或村口灵灯时，当前目标出现统一高亮。
- 灵石在可调查阶段出现冷色微光。
- 灵灯在任务完成后出现稳定暖色光晕。
- 左上任务提示显示中文阶段文案，而不是内部阶段枚举名。
- 交互按钮、摇杆和对话框在横屏下不互相遮挡。
